Installation Guide

Getting Fridge running on your Mac.

By Nico Crisafulli

Installation

Three steps.

1

Open the file you received.

Double-click Fridge.dmg. A small window appears with the Fridge icon and an Applications folder shortcut.

2

Drag Fridge into Applications.

Click and hold the Fridge icon, drag it onto the Applications folder, and let go. Mac copies the app in.

3

Close the installer window.

You can eject the DMG by dragging it to the trash, or right-click and Eject. The installation file is no longer needed.

Now you're ready to launch it. Mac will show a warning the first time — that's normal and walked through on the next page.

First launch

About that warning.

Because Fridge isn't sold through the Mac App Store, macOS shows a warning the first time you open it. This is normal. The app is safe — Mac just doesn't recognize the developer (me) yet. You'll only see this once.

1

Find Fridge in Applications.

Open Finder, click Applications in the sidebar, scroll to Fridge.

2

Right-click Fridge. Choose Open.

Important: right-click (or Ctrl-click) on the app icon — don't just double-click. A menu appears. Click Open.

3

Click Open in the dialog.

Mac asks if you're sure. Click the Open button. Fridge launches. From now on, double-clicking works normally — you're done with this dance forever.

Still won't open? Open System Settings → Privacy & Security. Near the bottom there's a line: "Fridge was blocked from use because it is not from an identified developer." Click Open Anyway.
